Shaketobug is in-app bug reporting library for android

Make getting bug-report/feedback for your app easy and fun.

                    
	@Override
	protected void onCreate(Bundle savedInstanceState) {
		super.onCreate(savedInstanceState);
		setContentView(R.layout.activity_main);
		ShaketobugManager.getInstance().startListening(this);
	}
    
    @Override
	protected void onDestroy() {
		super.onDestroy();
		ShaketobugManager.getInstance().stopListening(this);
	}

How to use

1- Add following code to onCreate() method of your activity to start listening for shake events.


ShaketobugManager.getInstance().startListening(this);

2- Add following code to onDestroy() method of your activity to stop listening for shake events and to prevent memory leaks.


ShaketobugManager.getInstance().stopListening(this);

3- Add FeedbackActivity inside Application tag and permission WRITE_EXTERNAL_STORAGE (if you dont have it already) outside of Application tag. 


<activity
    android:name="com.shaketobug.ui.FeedbackActivity"
    android:label="@string/title_activity_feedback" >
    <intent-filter>
        <action android:name="com.shaketobug.ui.FeedbackActivity" />

        <category android:name="android.intent.category.DEFAULT" />
    </intent-filter>
</activity>

<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />

4- You can also customize ui of the library and configuration with instance of ShaketobugConfig class. Here is sample one. Dont forget to update email address field to your support email


ShaketobugConfig config = new ShaketobugConfig();
config.setActionbarColor(Color.RED);
config.setActionbarTitle("Sample Title");
config.setEmailSubjectField("Bug Report From User");
config.setEmailToField("burakdede87@gmail.com");
config.setPencilColor(Color.YELLOW);
config.setUseDarkIcons(false);
config.setActionbarBackgrounDrawable(R.drawable.yellow_header_background);
// notify library to use this config instead of default one.
ShaketobugManager.getInstance().setConfig(setupConfig());
